# School_District_Analysis
## Overview
The purpose of this School District Analysis is to show the different schools and each students grade based on the subject and what grade they are in. The Analysis also includes the budget and spending plan for the different districts. The dataframes clearly visualizes this data based on the index which in this case is the student_id. The student_id serves as a perfect index because it is a unique identifier for each student across the district. The analysis has different portions that categorize the scores by school size and school spending.
